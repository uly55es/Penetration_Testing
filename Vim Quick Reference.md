# Vim

| **Command** | **Function** |
| --- | --- |
| `vim file` | open file with vim |
| <kbd>Esc+I</kbd> | enter insert mode |
| <kbd>Esc</kdb> | back to normal mode |
| `x` | Cut character |
| `dw` | Cut word |
| `dd` | Cut full line |
| `yw` | Copy word |
| `yy` | Copy full line |
| `p` | Paste |
| `:1` | Go to line number 1 |
| `:w` | Write the file 'i.e. save' |
| `:q` | Quit |
| `:q!` | Quit without saving |
| `:wq` | Write and quit |
